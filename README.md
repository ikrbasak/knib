# knib

A pnpm + Turborepo monorepo for the `@knib/*` family of small, focused
TypeScript packages.

## Packages

| Package                             | Description                                                              |
| ----------------------------------- | ------------------------------------------------------------------------ |
| [`@knib/dotenv`](./packages/dotenv) | Opinionated `dotenv` wrapper that loads layered env files by `NODE_ENV`. |
